# SuperZIP demo

**NOTE:** Requires Shiny 0.9.0 or later.

You can run this demo with:
```
if (!require(devtools))
  install.packages("devtools")
devtools::install_github("jcheng5/leaflet-shiny")
shiny::runGitHub("superzip", "jcheng5", ref="march2014")
```
